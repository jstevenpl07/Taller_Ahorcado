# Taller_Ahorcado

Este proyecto es para hacer un minijuego como el ahorcado, utilizando los arreglos y matrices para usar ene el codigo.

## **Pasos para mostrar el uso del codigo**
- Importar el uso de las matrices
- Importar el uso de las variables
- Usar el ascii art para el muñeco
- Usar el archivo CSV para leer el archivo
