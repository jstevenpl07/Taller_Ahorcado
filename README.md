## 🎮 Juego del Ahorcado en Java
# 📌 Descripción

Este proyecto consiste en la implementación del clásico juego del Ahorcado, desarrollado en Java. 
El objetivo del juego es adivinar una palabra oculta letra por letra antes de que el personaje sea completamente dibujado (y pierda la partida).

# 🚀 Características
Selección de palabras
Validación de letras ingresadas
Contador de intentos fallidos
Representación del progreso del jugador
Mensajes de victoria o derrota
Lógica del "muñeco" que se completa con cada error

# 🧠 Conceptos aplicados

Durante el desarrollo de este proyecto se utilizaron:

Estructuras de control (if, while, for)
Arreglos o listas
Manejo de cadenas (String)
Entrada de datos por consola
